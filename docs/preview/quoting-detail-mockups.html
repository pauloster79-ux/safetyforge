How a number comes to be

Kerf's value isn't that it generates a quote. Every estimating tool does that. Kerf's value is every number is citable. The contractor can ask "where did $9,800 come from?" and the chain resolves: WorkItem 04 → 12 floor boxes × ($380 labour + $270 items) × 22% margin, with labour adjusted +15% by the low-ceiling insight derived from Alder St, Chapelwood, Peachtree.

This document shows the screens that surface that chain. Running example: Maple Ridge Phase II · WorkItem 04 · Floor boxes — the number most contractors get wrong. Screens show progressively deeper drills: quote overview → WorkItem inspector → labour line detail → item line detail → methodology cascade → source provenance.
